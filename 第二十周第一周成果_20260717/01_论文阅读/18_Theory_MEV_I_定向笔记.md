# 18《Theory of MEV I》定向笔记

> 来源：`文献收集/18_Theory_MEV_I.pdf`，PDF 共 22 页。本周只抽取与“最小可利用性见证”有关的 AMM 形式化；不把恒定乘积 AMM 的最优攻击结论外推到 Arbitrum 跨层窗口。

## 一句话结论

论文将 AMM、用户钱包和待执行交易建模为状态转换系统，再允许拥有排序、插入和删除能力的对手构造交易序列，以对手净资产变化定义 MEV；它说明“窗口”只有在状态、可行动作、有效序列和净收益齐备时才可能成为可利用性证据。（PDF p.2–6，Sec. 1–3）

## 论文中值得保留的形式化元素

| 元素 | 论文定义/作用 | 对本项目的保留方式 |
|---|---|---|
| 前置状态 `Γ` | 钱包余额与 AMM 储备共同构成执行前状态。（PDF p.3） | 后续若研究某个 DEX 行为，需保存可重建的 L2 pre-state；第 1 周尚未做。 |
| 用户交易池 `X` | 待执行用户交易的有限多重集合。（PDF p.5） | 不能用历史 block 反推完整 Sequencer txpool；故不把它当已观测对象。 |
| 对手动作 | 构造包含部分用户交易和任意对手交易的有序序列，可重排、插入或删除。（PDF p.5） | 仅在明确某角色确有该权限时使用；普通 searcher 默认没有删除/重排他人交易权。 |
| 有效执行 | 序列必须在 `Γ` 上可执行；不可执行序列的 gain 为零。（PDF p.5–6） | 候选行为必须检查 receipt/trace/状态约束，不能只看位置。 |
| 净收益 | 以执行前后钱包净值差 `G_A(Γ, λ)` 表达。（PDF p.6，Eq. 2–3） | 实务中需加入 gas、协议费和统一价格口径；未计算时不称“利润”。 |

## 最小“可利用性见证”模板

```text
前置状态 Γ
  → 可观察信息 I（含可见时间锚点）
  → 角色 R 的允许动作 A（排序/插入/延迟的权限来源）
  → 原始顺序 λ_original
  → 反事实顺序 λ_counterfactual
  → 两者是否有效执行、receipt/trace 与状态变化
  → 统一计价下的净收益 ΔV（扣 gas、协议费、失败成本）
  → 证据等级：候选 / 可执行 / 反事实收益
```

第 1 周仅产生 `I` 中的 L1 入块和 L2 creation/redeem 时间关系；没有重建 `Γ`、`λ_counterfactual` 或 `ΔV`，因此不形成可利用性见证。

## 重要局限

论文明确假设固定原子资产价格、无交易费、无 swap fee、交易序列长度不受限等，且在讨论部分说明这些假设影响最优策略和收益。（PDF p.6，Sec. 3 假设；p.9–10，Sec. 6）因此其数值例子和最优性证明不能被用来估计本项目的真实利润。
