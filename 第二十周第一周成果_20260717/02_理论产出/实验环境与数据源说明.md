- 实验性质：预实验 / 数据链路验证，非正式实验。

## 可复现性控制

- L1 finalized block hash、对象 hash、合约 code hash、batch/message 样本量和参数均保存。
- 代码把 raw、processed、figures 和日志分目录保存；RPC token 只允许环境变量读取。
- batch 映射使用 NodeInterface 单调二分，message 映射使用官方 SDK 同构 ticket ID，均不采用时间最近配对。
- 运行日志记录本次最终运行：70 次 L1 RPC 调用、737 次 L2 RPC 调用；初次大整数 JSON 精度失败保留在 `07_运行日志/run_error.log`，随后修复为字符串传递并成功完成最终运行。
