# 证据等级与结论边界

## 固定证据链

```text
E1 时间关系 → E2 公开可见性 → E3 行动能力 → E4 候选行为 → E5 反事实利润
```

| 等级 | 最小证据 | 本周是否可达 | 禁止的升级表述 |
|---|---|---|---|
| E1 时间关系 | 固定对象映射、block hash/timestamp、可复算时间差 | 是 | “存在攻击” |
| E2 公开可见性 | 公开 L1 入块或可审计事件 | 部分；仅 L1 入块操作口径 | “已知所有人 first-seen 时间” |
| E3 行动能力 | 角色权限、提交路径、资金/执行可行性 | 仅理论边界 | “已能利用” |
| E4 候选行为 | 预注册标签、链上字段、人工复核 | 否 | “Sequencer 偏置/攻击频率” |
| E5 反事实利润 | 前态、替代顺序、有效执行、成本扣除与价值计量 | 否 | “已实现 MEV 利润” |

## 必须保留的否定句

- 窗口存在 ≠ 存在可利用 MEV。
- 交易位置靠前 ≠ Sequencer 排序偏置。
- 地址集中 ≠ 实体集中。
- revert 高 ≠ 机制执行质量差。
- 观察到候选行为 ≠ 已确认攻击。
- Timeboost 的拍卖/express-lane 指标 ≠ 跨层 MEV 指标。

## 第 1 周结论上限

本周最多报告“对 selected finalized L1 range 中的 eligible retryable message，L1 入块与 L2 creation/redeem 对象可以在何种比例下精确映射，及其时间差的描述性分布”。任何关于利用、偏置、利润或防御效果的主张均超出本周证据。
